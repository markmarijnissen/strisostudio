# MidiOutput

Convert Striso events into MIDI messages and send them to a MIDI output device.

## Attributes

- `input`: Incoming Striso Messages

## Example

```html
<d-compose output="striso"></d-compose>
<midi-output input="striso"></midi-output>
```

<d-compose output="striso"></d-compose>
<midi-output input="striso"></midi-output>
